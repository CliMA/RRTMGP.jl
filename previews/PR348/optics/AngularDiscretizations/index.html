Angular Discretizations

RRTMGP.AngularDiscretizations.AngularDiscretization — Type
AngularDiscretization{FT,FTA1D,I}

Weights and angle secants for first order (k=1) Gaussian quadrature. Values from Table 2, Clough et al, 1992, doi:10.1029/92JD01419 after Abramowitz & Stegun 1972, page 921

Fields

  • n_gauss_angles: number of quadrature angles

  • gauss_Ds: quadrature secants / secant of propagation angle

  • gauss_wts: quadrature weights
