Atmospheric State

RRTMGP.AtmosphericStates.AtmosphericState — Type
AtmosphericState{FT,FTA1D,FTA1DN,FTA2D,CLDP,CLDM,VMR,I} <:
    AbstractAtmosphericState{FT,I,FTA1D}

Atmospheric conditions, used to compute optical properties.

Fields

  • lon: longitude, in degrees (ncol), optional

  • lat: latitude, in degrees (ncol), optional

  • p_lay: Layer pressures [Pa, mb]; (nlay,ncol)

  • p_lev: Level pressures [Pa, mb]; (nlay+1,ncol)

  • t_lay: Layer temperatures [K]; (nlay,ncol)

  • t_lev: Level temperatures [K]; (nlay+1,ncol)

  • t_sfc: Surface temperatures [K]; (ncol)

  • col_dry: Number of molecules per cm^2 of dry air (nlay, ncol)

  • vmr: volume mixing ratio of all relevant gases

  • cld_r_eff_liq: effective radius of cloud liquid particles

  • cld_r_eff_ice: effective radius of cloud ice particles

  • cld_path_liq: cloud water path

  • cld_path_ice: cloud ice path

  • cld_frac: cloud fraction

  • random_lw: random number storage for longwave bands (ngpt, nlay, ncol)

  • random_sw: random number storage for shortwave bands (ngpt, nlay, ncol)

  • cld_mask_lw: cloud mask (longwave), = true if clouds are present

  • cld_mask_sw: cloud mask (shortwave), = true if clouds are present

  • cld_mask_type: cloud mask type

  • ice_rgh: ice roughness, 1 = none, 2 = medium, 3 = rough

  • nlay: Number of layers.

  • ncol: Number of columns.

  • ngas: Number of gases.

RRTMGP.AtmosphericStates.GrayAtmosphericState — Type
GrayAtmosphericState{FT,FTA1D,FTA2D,I} <: 
    AbstractAtmosphericState{FT,I,FTA1D}

Atmospheric conditions, used to compute optical properties with the gray atmosphere approximation

Fields

  • lat: latitude, in degrees, for each column; (ncol,)

  • p_lay: Layer pressures [Pa, mb]; (nlay, ncol)

  • p_lev: Level pressures [Pa, mb]; (nlay+1, ncol)

  • t_lay: Layer temperatures [K]; (nlay, ncol)

  • t_lev: Level temperatures [K]; (nlay+1,ncol)

  • z_lev: Level Altitude [m]; (nlay+1,ncol)

  • t_sfc: Surface temperatures [K]; (ncol)

  • otp: optical thickness parameters

  • nlay: Number of layers.

  • ncol: Number of columns.
