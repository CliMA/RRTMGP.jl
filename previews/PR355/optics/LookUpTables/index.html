RRTMGP.LookUpTables.LookUpCld — Type
LookUpCld{I,FT,FTA1D,FTA2D,FTA3D,FTA4D}

Lookup table for cloud optics.

This struct stores the tables and Pade coefficients for determing extinction coeffient, single-scattering albedo, and asymmetry parameter g as a function of effective radius. We compute the optical depth tau (=exintinction coeff * condensed water path) and the products taussa and taussa*g for liquid and ice cloud separately. These are used to determine the optical properties of ice and water cloud together.

Fields

  • nband: number of bands

  • nrghice: number of ice roughness types

  • nsize_liq: number of liquid particle sizes

  • nsize_ice: number of ice particle sizes

  • nsizereg: number of size regimes

  • ncoeff_ext: number of extinction coefficients for pade approximation

  • ncoeff_ssa_g: number of ssa/g coefficients for pade approximation

  • nbound: number of size regime boundaries for pade interpolation

  • pair: pair = 2

  • radliq_lwr: liquid particle size lower bound for LUT interpolation

  • radliq_upr: liquid particle size upper bound for LUT interpolation

  • radliq_fac: factor for calculating LUT interpolation for liquid particle

  • radice_lwr: ice particle size lower bound for LUT interpolation

  • radice_upr: ice particle size upper bound for LUT interpolation

  • radice_fac: factor for calculating LUT interpolation for ice particle

  • lut_extliq: LUT liquid extinction coefficient (nsize_liq, nbnd) m²/g

  • lut_ssaliq: LUT liquid single scattering albedo (nsize_liq, nbnd)

  • lut_asyliq: LUT liquid asymmetry parameter (nsize_liq, nbnd)

  • lut_extice: LUT ice extinction coefficient (nsize_ice, nband, nrghice) m²/g

  • lut_ssaice: LUT ice single scattering albedo (nsize_ice, nband, nrghice)

  • lut_asyice: LUT ice asymmetry parameter (nsize_ice, nband, nrghice)

  • pade_extliq: pade coefficients for liquid extinction (nband, nsizereg, ncoeff_ext)

  • pade_ssaliq: pade coefficients for liquid single scattening albedo (nband, nsizereg, ncoeff_ssa_g)

  • pade_asyliq: pade coefficients for liquid asymmetry paramter (nband, nsizereg, ncoeff_ssa_g)

  • pade_extice: pade coefficients for ice extinction (nband, nsizereg, ncoeff_ext, nrghice)

  • pade_ssaice: pade coefficients for ice single scattening albedo (nband, nsizereg, ncoeff_ssa_g, nrghice)

  • pade_asyice: pade coefficients for ice asymmetry paramter (nband, nsizereg, ncoeff_ssa_g, nrghice)

  • pade_sizreg_extliq: pade size regime boundaries for liquid extinction coefficient for pade interpolation (nbound) μm

  • pade_sizreg_ssaliq: pade size regime boundaries for liquid single scattering albedo for pade interpolation (nbound) μm

  • pade_sizreg_asyliq: pade size regime boundaries for liquid asymmetry parameter for pade interpolation (nbound) μm

  • pade_sizreg_extice: pade size regime boundaries for ice extinction coefficient for pade interpolation (nbound) μm

  • pade_sizreg_ssaice: pade size regime boundaries for ice single scattering albedo for pade interpolation (nbound) μm

  • pade_sizreg_asyice: pade size regime boundaries for ice asymmetry parameter for pade interpolation (nbound) μm

  • bnd_lims_wn: beginning and ending wavenumber for each band (2, nband) cm⁻¹

  • use_lut: use LUT (default) or pade interpolation
